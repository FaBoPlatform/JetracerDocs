JetCard

JetPack4.3に対応したJetCardで基本環境を作成します。

JetCardをインストール

JetCardをCloneします。

1
2
cd $HOME
git clone https://github.com/NVIDIA-AI-IOT/jetcard

jetpack_4.3ブランチに切り替えます。

1
2
cd jetcard
git checkout jetpack_4.3

JetCardをインストールします。

1
./install.sh
