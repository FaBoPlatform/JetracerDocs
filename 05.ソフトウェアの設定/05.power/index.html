    電力モード

    Jetson Nanoでは、2つの電力モードが存在しています。

    モード コマンド 使用CPU
    5W sudo nvpmodel -m 1 2個
    MAXN sudo nvpmodel -m 0 4個

    電力モードを変える事で、CPUの使用数を変える事が可能になります。

    5W Powerモードの設定

    5Wモードに変更します。

    1
    sudo nvpmodel -m 1

    電力モードが変更されたかの確認をおこないます。

    1
    nvpmodel -q --verbose

    Tip

    sudo nvpmodel -m 1 が省電力モードになり、sudo nvpmodel -m 0が標準電力モードになります。sudo nvpmodel -m 0では電力が4A程度必要になります。
