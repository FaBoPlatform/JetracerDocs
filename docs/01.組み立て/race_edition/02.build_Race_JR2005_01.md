# JetRacer Kit Race Edition JR2025シリーズ 組立て手順

## 【対象モデル】

|コード番号|名称|TAMIYA TT-02 XB|
|:--|:--|:--:|
|JR2025-R|FaBo JetRacer Kit Race Edition|◯|
|JR2025-R-NV|FaBo JetRacer Kit Race Edition 車体なし|X|

!!!Cation "六角丸ビスの下穴"
	※六角丸ビスM3X10ご使用の際は、3mmビス穴タップ(M3×0.5)で下穴の処理をしてください。

## RCカーのコンポーネント(本キットの場合)

タミヤ TT-02 エキスパートビルドのご使用の場合（通常版）

受信機を取り外し、ねじを４本取り外します。

## RCカーのコンポーネントのご紹介

タミヤ TT-02 SRXまたはタミヤ TT-02のご使用の場合、サーボ、ESC取り付けまたは交換例(FaBo 2024年度のレース用車体）

| | |
|:--|:--|
|ステアリングサーボ|SANWA PGS-CL Ⅱ|
|スピードコントローラー|HOBBYWING QuicRUN-WP-1060-Brushed BEC内蔵3A/6V(HW 30120201)|
|ブラシモーター|OP.779 GTチューンモーター（25T）|
|SRXの場合、デフを変更|XV-02/TT-02 オイル封入式ギヤデフ (39T) ユニット|

※上のリスト部品は、本キットには付属しません。

![](./img/001assy/JR2025R002.JPG)

なべねじM3*10を４本、M3ナットでフロントマウントをカーボンベースに取り付けします。

![](./img/001assy/JR2025R003.JPG)

ナットドライバーと＋ドライバーでねじを緩まないよう締めます。

![](./img/001assy/JR2025R004.JPG)

![](./img/001assy/JR2025R005.JPG)

皿ねじM3X10とナットM3それぞれ４個でJetsonベースを取り付けます。

![](./img/001assy/JR2025R006.JPG)

![](./img/001assy/JR2025R007.JPG)

スパナ5.5と＋ドライバーを使用して締め付けます。

![](./img/001assy/JR2025R008.JPG)

バックマウントを取り付けます。皿ねじM3*12 ４本、ナットM3 ４本用意します。

![](./img/001assy/JR2025R009.JPG)

![](./img/001assy/JR2025R010.JPG)

ナットドライバー5.5と＋ドライバーでねじを締め付けます。

![](./img/001assy/JR2025R011.JPG)

なべねじM3X10　２本、M3ナット　2個を用意します。

![](./img/001assy/JR2025R012.JPG)

リアバンパーの裏側からM3ナットを左右それぞれ入れます。

![](./img/001assy/JR2025R013.JPG)

カメラベースをリアパンパーに装着し、ねじを締めます。

![](./img/001assy/JR2025R014.JPG)

タッピングビスM3*10も左右それぞれ取り付けます。

![](./img/001assy/JR2025R015.JPG)

TT-02エキスパートビルドの装着されていたタッピングねじを使いRCカーに取り付けます。

![](./img/001assy/JR2025R016.JPG)

カメラマウントを組立て、取り付けます。六角丸ビスを使用します。

![](./img/001assy/JR2025R017.JPG)

六角丸ビスを写真の箇所に取り付けします。（きつい場合は、下穴の処理をしてください。）

![](./img/001assy/JR2025R018.JPG)

六角棒レンチで締め付けします。

![](./img/001assy/JR2025R019.JPG)

カラーLEDを取り付けます。

![](./img/001assy/JR2025R021.JPG)

六角棒レンチでねじを４箇所取り付けます。

![](./img/001assy/JR2025R022.JPG)

各2個のCSIカメラとケーブルを取り付けます。

![](./img/001assy/JR2025R023.JPG)

CSIカメラにCSIケーブルをコネクタに差し込み、抑え部で接続します。ケーブルの接点は下向きになります。

![](./img/001assy/JR2025R024.JPG)

ケーブルをカメラマウントへ通します。

![](./img/001assy/JR2025R025.JPG)

CSIカメラをねじで固定します。

![](./img/001assy/JR2025R026.JPG)

もう一つのカメラを固定します。

![](./img/001assy/JR2025R027.JPG)

カメラホルダーにカメラケーブルを通します。

![](./img/001assy/JR2025R028.JPG)

ケーブルホルダーにもCSIカメラケーブルを通します。

![](./img/001assy/JR2025R029.JPG)

カメラホルダーを固定します。

![](./img/001assy/JR2025R030.JPG)

CSIカメラを固定します。

![](./img/001assy/JR2025R031.JPG)

SPIーLEDのケーブル（長）を写真右側に接続します。

![](./img/001assy/JR2025R032.JPG)

リアのSPI-LEDを接続します。

![](./img/001assy/JR2025R033.JPG)

SPI-LEDケーブル短い方を用意します。SPI-LED基板には、OUTPUT ,INPUTがありご注意ください。

![](./img/001assy/JR2025R034.JPG)

写真のようにSPI-LEDケーブルを接続します。

![](./img/001assy/JR2025R035.JPG)

ケーブル２つのうち、どちらかをリアマウントにある穴にカメラのケーブルを１つだけ通します。

![](./img/001assy/JR2025R036.JPG)

SPI-LED基板をねじで固定します。

![](./img/001assy/JR2025R037.JPG)

SPL-LEDケーブルを写真のようにバックマウントの下側を通し、フックに掛けます。カメラマウントをカメラベースに取り付けます。

![](./img/001assy/JR2025R038.JPG)

反対側も同じようにカメラマウントをカメラベースに取り付けます。

![](./img/001assy/JR2025R039.JPG)

マジックバンドをリアマウントにある穴に通します。

![](./img/001assy/JR2025R040.JPG)

カメラマウントを支持するパーツを取り付けます。SRX用とXB用があります。（写真はSRX）

![](./img/001assy/JR2025R041.JPG)

カメラ固定パーツをRCカーのダンパープレートと取り付けます。

![](./img/001assy/JR2025R042.JPG)

反対側も同じくカメラ固定パーツをRCカーのダンパープレートと取り付けます。

![](./img/001assy/JR2025R043.JPG)

Jetson Orin Nanoを取り付けます。マイクロSDカードをあらかじめJetson Orin NanoのSDカードスロットに挿入しておきます。マイクロSDカードはOSが入っていない場合は、ダウンロードセンターからSDカードイメージをダウンロードしSDカードにインストールしてください。

![](./img/001assy/JR2025R044.JPG)

CAM0のケーブルは上側のカメラ、CAM1のケーブルは下側のカメラに接続してください。

Jetson Orin NanoをJetsonベースに載せ、CSIカメラケーブルをJetsonそれぞれCAM0,CAM1に接続します。コネクタはとても壊れやすくやさしく取扱いご注意願います。

![](./img/001assy/JR2025R045.JPG)

コネクタの抑え部（黒い部分）は大変弱く、やさしく取り出し、ケーブルをコネクタに刺して再び戻してケーブルの接点と密着し取り付けます。

![](./img/001assy/JR2025R046.JPG)

透明なカバーで取り付けます。

![](./img/001assy/JR2025R047.JPG)

カバーを六角棒を使って、４箇所、ねじで取り付けます。

![](./img/001assy/JR2025R048.JPG)

CSIカメラケーブルをバッテリーの下にして取り付けます。

![](./img/001assy/JR2025R049.JPG)

両面テープを貼り付けます。

![](./img/001assy/JR2025R050.JPG)

外しておいた受信機を取り付けます。

![](./img/001assy/JR2025R051.JPG)

ずれて挿入しないように注意しながら、コントロール基板を取り付けます。

![](./img/001assy/JR2025R052.JPG)

SPI-LEDのケーブルをコントロール基板のSPI-LEDコネクタと接続します。

![](./img/001assy/JR2025R053.JPG)

基板にある文字　１ch、２ch、３chは、タミヤの受信機１、２、Bにそれぞれ繋ぎます。

Servoはサーボに、ESCはRCカーのESC(スピードコントローラーに繋ぎます。）

※極性を間違えないでください。（白線またはマークがあるもの：信号線、赤線：BEC電源、黒線：グラウンド）

![](./img/001assy/JR2025R055.JPG)

USBケーブルを結束します。

![](./img/001assy/JR2025R056.JPG)

ケーブルが路面に触れないようにケーブルは結束します。

![](./img/001assy/JR2025R065.JPG)

L字のUSBケーブル（Tyep-C）をバッテリーと接続します。

![](./img/001assy/JR2025R058.JPG)

トリガーゲーブルを接続します。

![](./img/001assy/JR2025R059.JPG)

車体の電源がOFFになっていることを確認してバッテリーを繋ぎます。

![](./img/001assy/JR2025R060.JPG)

安全を確認し車体を必ず作業台に乗せてから、ONします。

![](./img/001assy/JR2025R064.JPG)

トリガーケーブルのDCプラグをJetson Orin NanoのDCジャックへ接続します。

![](./img/001assy/JR2025R061.JPG)

送信機は、電源ONするときはステアリング、スロットルに触らずONしてください。

![](./img/001assy/JR2025R062.JPG)

 必ず車体を作業台に乗せて、RCモード（手動操作）とAIモード（自動操作）の切り替えボタンがあります。送信機の裏側ボタン押すとカラーLEDが変わることを確認してください。

![](./img/001assy/JR2025R063.JPG)

基本的な動作　プロポによる操作、信号切り替えが確認できた場合は、２つのカメラのレンズキャップを外してください。
以上組み立ては完成です。
