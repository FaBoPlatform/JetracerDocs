Multiplexer

Multiprexer

FaBo JetRacer #612では、Multiplexerのチップでプロポ制御、JetRacer制御を切り替えます。

A3のPinの状態をみて、A3が押されている場合は、D2のピンをHighに、押されていない場合は、Lowにします。

ピン 動作
D2 Multiplexerの切り替え
A3 SELの値の入力

void setup(){
  pinMode(2, OUTPUT);
  pinMode(3, OUTPUT);
  pinMode(4, OUTPUT);
  pinMode(A3, INPUT);
}

void loop(){
  int duration = pulseIn(A3, HIGH,25000);
  if(duration > 100){
    digitalWrite(2, HIGH);
    digitalWrite(3, HIGH);
    digitalWrite(4, LOW);
  } else{
    digitalWrite(2, LOW);
    digitalWrite(3, LOW);
    digitalWrite(4, HIGH);
  } 
}
